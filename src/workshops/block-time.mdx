# Block time for the workshop

Block the time needed for the workshop on the calendar of all participants. 

Make sure every member of your team, the deciders and the experts get invitations and send their acceptance.
Set a reminder for a day before the workshop when preparing the invites.

Make sure you don't forget to block time for yourself.
